Layanan

Pembuatan Website Statis

Rp 2.000.000 - Rp 5.000.000

Pembuatan website statis adalah proses pembuatan website yang tidak memerlukan database dan hanya menampilkan konten statis.

  • Desain website yang responsif dan modern
  • Integrasi dengan teknologi web terbaru
  • Pengujian dan debugging website

Pembuatan Aplikasi Web Dinamis

Rp 5.000.000 - Rp 20.000.000

Pembuatan aplikasi web dinamis adalah proses pembuatan aplikasi web yang memerlukan database dan dapat menampilkan konten dinamis.

  • Desain aplikasi web yang responsif dan modern
  • Integrasi dengan teknologi web terbaru
  • Pengujian dan debugging aplikasi web
  • Pengembangan fitur-fitur aplikasi web

Desain UI/UX

Rp 1.500.000 - Rp 3.000.000

Desain UI/UX adalah proses pembuatan desain antarmuka pengguna yang mudah digunakan dan menarik.

  • Desain antarmuka pengguna yang responsif dan modern
  • Pengembangan pengalaman pengguna yang baik
  • Pengujian dan debugging desain

Maintenance dan Update Website

Rp 500.000 - Rp 2.000.000 per bulan

Maintenance dan update website adalah proses pemeliharaan dan pembaruan website yang sudah ada.

  • Pengujian dan debugging website
  • Pembaruan konten website
  • Pengamanan website dari serangan
